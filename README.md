📌 Activity Description

This project is a responsive multi-page website created using Bootstrap 5, HTML5, CSS3, and JavaScript. It demonstrates the implementation of Bootstrap’s Grid System, responsive navigation bar, interactive components, and form validation with JavaScript.

🧩 Features

    - Responsive Bootstrap Grid System
    - Mobile-friendly navigation bar
    - Interactive Bootstrap Modal/Carousel
    - Functional form validation using JavaScript
    - Responsive layouts using Flexbox and Grid
    - Mobile-first responsive design
    - Organized project file structure
